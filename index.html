Git In a nutshell

Git is a version control system for tracking changes in computer files and coordinating work on those files among multiple people. As a distributed revision control system, it is aimed at speed, data integrity, and support for distributed, non-linear workflows. Source:Wikipedia. The following video explains the fundamentals very clearly. This video is a command-line git video. For a GUI git tutorial check youtube.







What Next ?

Try learning advanced git commands. You should also open a GitHub account - it's free! If you are ever in need of help - just contact us ! We will do out best to sort out your problems.
